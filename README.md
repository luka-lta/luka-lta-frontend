## ⚙️ Setup & Installation

### Prerequisites

- **Node.js** `>= 18.x`
- **npm** or **pnpm**

### Run Locally

```bash
# Clone the repository
git clone https://github.com/luka-lta/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```

### Create a Build

```bash
# Production build
npm run build

# Preview the build locally
npm run preview
```
